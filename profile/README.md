## Welcome to the AI4HEP-UK Project Organisation 👋

Working in the UK on AI/ML topics, or interested in joining the efforts? We would love to hear from you!

**Here are some ideas to get you started:**

- 🙋‍♀️ Browse the [AI4HEP-UK website](https://ai.hep.ac.uk/) to learn about the project and activities.
- 🌈 For contributions, please get in touch with the team. We look forward to hosting relevant and interesting project repositories under our organisation.
- 📧 We have a AI4HEP-UK@jiscmail.ac.uk mailing list (subscribe [here](https://www.jiscmail.ac.uk/cgi-bin/wa-jisc.exe?A0=AI4HEP-UK)) and the CERN Mattermost channel https://mattermost.web.cern.ch/uk-hep/channels/ai4hep.
- 👩‍💻 Please check the GitHub repository [ai4hep-uk.github.io](https://github.com/AI4HEP-UK/ai4hep-uk.github.io) to contribute to the website.
